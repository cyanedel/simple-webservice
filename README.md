# Simple Web Service
A Simple REST service written in Java.


## Main Dependencies
```
  {
    BuildTool: Gradle 8.7,
    RestFramework: Spring Boot,
    Database: MySQL
  }
```

## Quick Command Guide
Clean the project
```
gradlew clean
```

Build the project
```
gradlew build
```

Start the Project
```
java -jar .\build\libs\simplews-0.1.0.jar
```
